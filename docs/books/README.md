# The SEO / AEO / GEO Book Series

A complete, practitioner-grade series covering search visibility from first principles through AI-generated answers.

## The Books

| Book | Chapters | Focus |
|---|---|---|
| [SEO Book](complete/seo/README.md) | 20 | Traditional search: crawling, indexing, ranking, on-page and technical SEO, local SEO, and analytics |
| [AEO Book](complete/aeo/README.md) | 10 | Answer Engine Optimization: ChatGPT, Google AI Overviews, Perplexity, Gemini, Claude, citations, and llms.txt |
| [GEO Book](complete/geo/README.md) | 8 | Generative Engine Optimization: knowledge graphs, entities, embeddings, vector search, and RAG |

## Reading Order

The books build on each other and are best read in sequence:

1. **SEO** establishes the foundation — a page must be crawlable, indexable, and well-optimized before anything else matters.
2. **AEO** extends that foundation into AI-powered answer engines — platform-specific tactics for getting cited.
3. **GEO** goes one layer deeper into the technology (knowledge graphs, embeddings, retrieval-augmented generation) that makes every AEO tactic work mechanically.

Each book cross-references the others throughout — see the companion book links at the top of each chapter's table of contents.
