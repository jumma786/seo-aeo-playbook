# 1. Introduction

Search engines help users discover information on the web by crawling websites, processing content, storing it in an index, and ranking pages based on hundreds of signals.

The process can be simplified into four major stages:

1. Discover
2. Crawl
3. Index
4. Rank

Understanding this workflow is the foundation of technical SEO and modern AI search optimization.

---

# 2. What is a Search Engine?

A search engine is a software system that discovers, stores, organizes, and retrieves web pages based on a user's query.

Examples include:

- Google
- Bing
- DuckDuckGo
- Brave Search
- Yahoo

Modern AI-powered search systems also include:

- ChatGPT Search
- Google AI Overviews
- Gemini
- Claude
- Perplexity
