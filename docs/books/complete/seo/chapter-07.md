# 1. Introduction

Search intent is the reason behind a user's search query. Understanding intent helps create content that satisfies user needs instead of simply targeting keywords. Modern search engines prioritize pages that best match the user's intent.

---

# 2. The Four Main Types of Search Intent

### Informational

The user wants to learn something.

Examples:

- What is SEO?
- How does Google rank websites?
- SEO checklist

### Navigational

The user wants to visit a specific website or page.

Examples:

- GitHub
- Google Search Console
- Ahrefs login

### Transactional

The user is ready to take action.

Examples:

- Buy SEO software
- Hire SEO consultant
- SEO agency near me

### Commercial Investigation

The user is comparing options before making a decision.

Examples:

- Best SEO tools
- Ahrefs vs Semrush
- Best keyword research software
